# shield-job

> 基于队列的分布式任务调度及分发中间件，based on RocketMQ

## 结构

| 项目 | 描述 | 功能 |
|  :------ |  :------ |  :------ |
| shield-job-common | shield任务调度核心 | 
| shield-job-scheduler-core | shield任务核心 |
| shield-job-example | shield任务调度示例 |

## 特性
1. 基于RocketMQ实现任务分发，可与定时调度进行无缝集成
2. 解析发送返回体，业务层做后续处理等
